### Start app locally

#### Setup

You need Node and Npm version specified in [package.json](./package.json) or use `nvm`
```
nvm use
```

Install dependencies
```
npm i
```

#### Run UI

```
npm run ui
```

#### Build UI

```
npm run build:ui
```

#### Run app

```
npm run build:ui
npm run app
```

#### Run build
```
npm run build
```
